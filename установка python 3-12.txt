Установка python
winget install Python.Python.3.12

Создание виртуального окружения 3.12
py -3.12 -m venv venv

Запуск
.\venv\Scripts\activate

Установка зависимостей (с зеркалом, если pypi.org недоступен)
python -m pip install -r .\requirements.txt -i https://mirrors.aliyun.com/pypi/simple/ --trusted-host mirrors.aliyun.com --timeout 120 --retries 10
